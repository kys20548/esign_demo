# 行動簽核 e-Sign Demo

機電工程場景的行動簽核系統雛型：工程師在案場用手機填寫「配電盤檢測簽核單」（含拍照上傳），主管在後台即時核准／退回，派工清單一眼看出**哪些客戶還沒去簽**。

純 Go 標準庫實作、零外部依賴，`go build` 產出單一執行檔即可部署。

## 功能

- **我的待簽客戶**：派工清單依狀態排序（未填單 → 待簽核 → 已核准/已退回），可依工程師篩選；點選未填單的派工，表單自動預填客戶、案場、盤體與人員
- **行動填單**：手機瀏覽器直接使用（RWD），檢測項目勾選、備註、現場拍照上傳（`capture` 直接叫出相機）
- **簽核後台**：待簽清單一鍵核准／退回，處理紀錄留存，狀態同步回派工清單

## 技術要點

- Go 標準庫：`net/http`（1.22+ 路由 pattern）、`html/template`、`embed`（模板與靜態檔編進執行檔）
- 資料存於記憶體（demo 定位），每小時自動重置為種子資料
- 監聽埠讀取 `PORT` 環境變數（雲平台相容），預設 8080

## 執行

```bash
go run .
```

啟動後終端機會印出本機與區網網址，手機連同一 Wi-Fi 即可開啟。

## 部署

附 `Dockerfile`（多階段建置，distroless 執行）與 `render.yaml`（Render 免費方案藍圖），亦可直接：

```bash
CGO_ENABLED=0 go build -o esign .
./esign
```
